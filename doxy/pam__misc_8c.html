Set the login time to the given pointer
Format: a b d H:M:S Y
Example: Fri Feb 11 02:59:18 2020

b: locale's abbreviated month name (e.g., Jan)
d: day of month (e.g., 01)
H: hour (00..23)
M: minute (00..59)
